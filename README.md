
## Setup

### Backend
1. `cd backend`
2. `cp env.example .env` and fill in values
3. `npm install`
4. `npm start`

### Frontend
1. `cd frontend`
2. `cp .env.example .env` and fill in values
3. `npm install`
4. `npm run dev`

## Features
- Auth (Sign Up, Sign In, etc.)
- Blog CRUD
- News
- File Management

## Environment Variables
- See `.env.example` in each folder.
